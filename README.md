
# About

[第2回 Japan Xamarin User Group Conference 東日本編 : ATND](https://atnd.org/events/57246 "第2回 Japan Xamarin User Group Conference 東日本編 : ATND") で発表した「Xamarin.Formsの概要とカスタムコントロール」のサンプルプログラムです。スライドは[Xamarin.formsとカスタムコントロールの話](http://www.slideshare.net/moonmile/xamarinforms-41882310 "Xamarin.formsとカスタムコントロールの話") にあるので、合わせてご覧ください。

サンプルは、

- カスタムコントロールを作る SampleButtonPage.xaml
- DependencyService を使う SampleDependency.xaml 
- カスタムレンダラを使う SampleMyRenderer.xaml

となっています。カスタムコントロールは、Grid を継承して複数のボタンや画像を一括で扱う方法、カスタムレンダラは Label コントロールにタップイベントを付ける方法が入っています。
